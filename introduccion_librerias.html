Introduccion a las Librerias

Las librerias en Python son colecciones de funciones y metodos predefinidos que se pueden utilizar para realizar tareas especificas sin tener que escribir el codigo desde cero.

Importacion de Librerias

Para utilizar una libreria en Python, primero se debe importar utilizando la palabra clave import. Ejemplo:

import math

print(math.sqrt(16))  # Imprime 4.0

Librerias Estandar

Python viene con una amplia variedad de librerias estandar que cubren areas como matematicas, manejo de archivos, networking, entre otros.

Librerias Externas

Ademas de las librerias estandar, Python cuenta con un vasto ecosistema de librerias externas desarrolladas por la comunidad para tareas especializadas como visualizacion de datos, machine learning, web scraping, etc.

Instalacion de Librerias Externas

Para instalar una libreria externa, se puede utilizar la herramienta pip (Python Package Index). Ejemplo:

pip install nombre_libreria
