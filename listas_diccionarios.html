Listas

Una lista en Python es una coleccion ordenada de elementos. Los elementos de una lista pueden ser de diferentes tipos de datos.

Definicion de Listas

Para definir una lista, se utilizan corchetes [] y se separan los elementos con comas. Ejemplo:

numeros = [1, 2, 3, 4, 5]

Acceso a Elementos de Lista

Los elementos de una lista se pueden acceder mediante indices. El primer elemento tiene indice 0. Ejemplo:

print(numeros[0])  # Imprime 1

Operaciones con Listas

Python proporciona varios metodos para trabajar con listas, como agregar elementos, eliminar elementos, ordenar elementos, etc.

Diccionarios

Un diccionario en Python es una coleccion no ordenada de pares clave-valor.

Definicion de Diccionarios

Para definir un diccionario, se utilizan llaves {} y se separan los pares clave-valor con comas. Ejemplo:

persona = {"nombre": "Juan", "edad": 30, "ciudad": "Mexico"}

Acceso a Elementos de Diccionario

Los elementos de un diccionario se pueden acceder mediante sus claves. Ejemplo:

print(persona["nombre"])  # Imprime "Juan"

Operaciones con Diccionarios

Los diccionarios en Python soportan operaciones como agregar elementos, eliminar elementos, y actualizar valores de elementos existentes.
