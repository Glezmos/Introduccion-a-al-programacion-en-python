Introduccion a Python

Python es un lenguaje de programacion interpretado, de alto nivel y de proposito general. Creado por Guido van Rossum y lanzado por primera vez en 1991, Python tiene una filosofia de diseno que enfatiza la legibilidad del codigo con el uso de una sintaxis notablemente clara.

Caracteristicas de Python

  • Facil de aprender y usar
  • Lenguaje interpretado
  • Lenguaje de proposito general
  • Extensible e integrable
  • Gran comunidad de usuarios

Primer Programa en Python

Vamos a escribir nuestro primer programa en Python:

print("Hola, mundo!")

Este simple programa imprime "Hola, mundo!" en la pantalla.

Instalacion de Python

Para comenzar a programar en Python, primero necesitas instalarlo en tu computadora. Puedes descargar la version mas reciente de Python desde el sitio oficial: python.org. Sigue las instrucciones de instalacion para tu sistema operativo.

Uso del Interprete de Python

Una vez instalado, puedes abrir el interprete de Python. Abre una terminal o simbolo del sistema y escribe python o python3 para iniciar el interprete interactivo. Veras un prompt como este:

>>

En el prompt, puedes escribir comandos de Python y ver los resultados de inmediato. Por ejemplo:

>> print("Hola, mundo!")
Hola, mundo!

Editores de Codigo

Para escribir programas mas largos, es recomendable usar un editor de codigo o un entorno de desarrollo integrado (IDE). Algunos editores populares son:

  • Visual Studio Code
  • Sublime Text
  • PyCharm
  • Atom

Ejecucion de Programas

Para ejecutar un programa de Python guardado en un archivo, guarda tu codigo en un archivo con extension .py (por ejemplo, programa.py). Luego, en la terminal, navega hasta el directorio donde guardaste el archivo y ejecuta:

python programa.py
